# Study-Notes
存放本人所有的学习笔记（电子档）

## 格式

所有笔记大致分为两类：

1. 思维导图类：后缀名`.xmind`，建议使用相应的应用程序打开

2. 电子书类：文件夹下有文件`book.toml`，且文件后缀名大部分以`.md`为主。推荐使用[mdBook](https://github.com/rust-lang/mdBook)构建，构建命令：

```bash
mdbook build
```
